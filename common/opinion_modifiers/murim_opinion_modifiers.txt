# Murim Chronicles - Martial World Opinion Modifiers
# VANILLA SEPARATION PRINCIPLE: These are MARTIAL-ONLY opinion modifiers.
# They exist in a PARALLEL layer to vanilla CK3 opinion.
# Martial respect does NOT affect feudal opinion, vassal loyalty, or political alliances.
# Martial rivalry does NOT cause wars, break alliances, or affect succession.
# These modifiers affect ONLY: martial interactions, clan politics, duel acceptance,
#   training willingness, tournament seeding, recruitment decisions.
# A character can HATE you politically but RESPECT you martially, and vice versa.
# INTEGRATION: Used by ALL martial event files for opinion changes
# INTEGRATION: murim_clan_scout_events.txt, murim_clan_recruitment_events.txt,
#              murim_sect_politics_events.txt, murim_duel_events.txt,
#              murim_youth_tournament_events.txt, murim_ranking_events.txt
# GENDER: No gender-based opinion modifiers. Respect is earned, not gendered.

# ============================================================
# VANILLA SEPARATION ENFORCEMENT NOTES
# ============================================================
# CRITICAL DESIGN RULE:
# None of these modifiers should EVER be used in:
# - Vassal opinion calculations
# - Alliance/marriage decision weights
# - War declaration triggers
# - Succession law evaluations
# - Council task effectiveness
# - Levy/tax calculations
#
# These modifiers exist in the MARTIAL LAYER ONLY.
# They affect:
# - Duel acceptance/refusal weights
# - Training event outcomes
# - Recruitment success chances
# - Tournament seeding and matchups
# - Clan internal politics (votes, promotions, succession)
# - Scout behavior (which courts to visit, who to approach)
# - Martial travel event options
#
# A French duke who is a respected martial artist still pays taxes to his king.
# A Shaolin elder who serves as a Song Dynasty courtier still does his court duties.
# The martial world is PARALLEL, not integrated, with vanilla politics.
# Characters can CHOOSE to merge these worlds through specific event options,
# but the systems never force it.
