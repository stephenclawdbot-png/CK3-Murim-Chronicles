# 🗡️ The Murim Chronicles - CK3 Total Conversion Mod

**All Under Heaven: The Way of the Martial Sage**

## Overview
A massive Crusader Kings 3 total conversion mod that transforms the medieval world into a wuxia/xianxia martial arts universe. Noble houses cultivate internal energy, master divine techniques, and wage endless war for control of the Murim - the martial world beneath heaven.

## Mod Scale
- **15,000+ lines** of code
- **200+ events**
- **50 martial sects/cultures**
- **300+ martial techniques**
- **100+ dynasty legacies**
- Fully compatible with "All Under Heaven"
- Text-heavy narrative focus

## Core Concept
In this world, the path to power is cultivation. Dynasty founders inscribed their understanding of the Dao into secret manuals passed to heirs. Techniques named after houses determine their destiny. The weak are devoured, the strong rule All Under Heaven.

## Features

### 🏯 Martial Houses System
Every dynasty possesses their namesake technique:
- **House Character** determines martial path (sword, palm, fist, etc.)
- **Cultivation Realms**: Qi Refinement → Foundation → Core → Nascent Soul → Void → Great Vehicle → Heavenly Tribulation
- **Martial Soul**: Each house manifests a spiritual form (Azure Dragon, White Tiger, Vermilion Bird, Black Tortoise, Qilin, Phoenix)

### ⚔️ Combat System
Three paths to dominance:
1. **Orthodox Sects** (Righteous Dao) - Public techniques, accepted cultivation
2. **Demonic Cults** (Heretical Dao) - Forbidden arts, blood sacrifices
3. **Unaffiliated Wanderers** (Free Dao) - Mixed heritage, unique fusion

### 📜 Technique System
```
Technique Rank: Mortal → Earth → Sky → Saint → Immortal → Divine
Categories:
- Body Cultivation (Iron Skin, Diamond Body)
- Sword Techniques (Azure Edge, Ten Thousand Swords)
- Palm Strikes (Descending Mountain, Shattering Void)
- Internal Energy (Azure Flame, Yin-Yang Balance)
- Movement Arts (Cloud Steps, Instant Transmission)
```

### 🌏 Territory - The Murim
Map regions transformed into cultivation zones:
- **Central Plains** - Orthodox sects, ancestral temples
- **Southern Wastes** - Demonic cults, forbidden grounds
- **Western Borders** - Foreign methods, mixed traditions
- **Eastern Seas** - Immortal islands, ancient beings
- **Northern Desolate Mountains** - Beast clans, primal energies

### 👥 Characters
- **Sect Masters**: Rulers holding ancient treasures
- **Elders**: Advisors with broken realms
- **Geniuses**: Young cultivators with heaven-defying talents
- **Crippled**: Fallen cultivators seeking revenge
- **Grand Elders**: Ancestors in seclusion

## Installation
1. Download ZIP or clone repo
2. Extract to `Documents/Paradox Interactive/Crusader Kings III/mod/`
3. Enable in launcher
4. Start new game with "The Murim Chronicles"

## Compatibility
- ✅ Base game 1.12.x
- ✅ All Under Heaven mod (recommended)
- ✅ Full text-heavy immersion
- ⚠️ Not ironman compatible (custom mechanics)
