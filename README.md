## Overview

**Murim Chronicles** transforms CK3 into a Wuxia/Xianxia cultivation world set in historical East Asia. Characters cultivate qi, practice martial techniques, join sects, duel rivals, undergo heavenly tribulations, and navigate the shadowy politics of the Jianghu -- the underground martial world that secretly controls the imperial court.

This mod integrates deeply with CK3's vanilla systems: council positions, wars, marriages, imprisonments, title grants, and vassal management are all influenced by the hidden cultivation world.

## Technical Architecture

### File Structure
```
CK3-Murim-Chronicles/
  common/
    ai_war_behavior/         -- Cultivation-aware military AI
    artifacts/               -- 45+ martial artifacts (9 categories)
    character_interactions/  -- Murim + Jianghu interactions
    character_modifiers/     -- 55+ combat/cultivation modifiers
    cultures/                -- Martial culture definitions
    decisions/               -- 21+ martial decisions (6 categories)
    dynasty_legacies/        -- 6 legacy tracks + 12 bloodline inheritances
    governments/             -- 8 sect government types
    lands/                   -- Asia Murim geographical setup
    on_actions/              -- 6 on-action files (AI, ranking, technique, jianghu)
    opinion_modifiers/       -- 100+ opinion modifiers
    religions/               -- Cultivation path religions
    schemes/                 -- 10+ martial schemes
    script_values/           -- 7 value calculation files
    scripted_effects/        -- 5 effect files (celestial, grind, jianghu, penalties, ranking)
    scripted_triggers/       -- 5 trigger files (conditions, grind, jianghu, ranking, techniques)
    traits/                  -- 6 trait files (90+ traits)
  events/
    16+ event files covering cultivation, duels, breakthroughs, artifacts,
    celestial events, combat, grind loops, imperial events, interactions,
    jianghu underground, rankings, sects, techniques, and regional events
  localization/english/
    6 localization files covering all mod content
```

### Design Principles
1. **Interrupt Safety**: All events use `exists = yes` checks before referencing variables/traits
2. **Graceful Degradation**: Missing variables/traits trigger fallback paths instead of errors
3. **AI Parity**: AI characters follow the same cultivation paths as players
4. **Interconnected Systems**: Cultivation, combat, alchemy, sects, and jianghu all cross-reference each other
5. **Variable Tracking**: Persistent variables track cultivation progress, duel records, sect membership, and reputation

## Installation

1. Clone or download this repository
2. Copy the mod folder to your CK3 mod directory:
   - **Windows**: `%USERPROFILE%\Documents\Paradox Interactive\Crusader Kings III\mod\`
   - **Linux**: `~/.local/share/Paradox Interactive/Crusader Kings III/mod/`
   - **Mac**: `~/Documents/Paradox Interactive/Crusader Kings III/mod/`
3. Copy `murim.mod` to the same mod directory
4. Enable "Murim Chronicles" in the CK3 launcher
