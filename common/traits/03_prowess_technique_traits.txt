################################################################
# MURIM CHRONICLES — PROWESS-GATED TECHNIQUE TRAITS
# Path: common/traits/03_prowess_technique_traits.txt
#
# DESIGN: Prowess is the universal requisite. Every technique
# trait requires minimum prowess PLUS one or more CK3 skills.
# Three tiers:
#   Tier 1 — Dual Stat (Prowess + 1 Skill)
#   Tier 2 — Triple Stat (Prowess + 2 Skills)
#   Tier 3 — Grand Master (Prowess + 3+ Skills, legendary)
#
# AI characters unlock these through yearly pulse events.
# Player characters unlock via decisions.
# Both use the same scripted triggers as gates.
################################################################
