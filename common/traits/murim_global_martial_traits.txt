#             murim_duel (foreign style duels), murim_ranking (foreign challenger), murim_clan_war (cross-faction)

#   Tang Poison ← Wudang purification / Shaolin Iron Body resistance

#   Berserker rage ← Tai Chi redirection / Wudang soft arts
#   Shaolin tank ← Tang poison attrition / Kalaripayattu pressure points
#   Speed (Iaijutsu/Engolo) ← Defensive (Wudang/Shaolin)
#   Mounted Mongol ← Terrain holders / formation fighters
